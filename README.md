# React Dashboard Example

## How to use

Download the example [or clone the repo](https://github.com/UVAMobileDev/NCAA-MODEL-REACT.git):

```sh
git clone https://github.com/UVAMobileDev/NCAA-MODEL-REACT.git
cd NCAA-MODEL-REACT
```

Install it and run:

```sh
npm install
npm install recharts
npm start
```
